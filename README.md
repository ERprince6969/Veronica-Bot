## ⚡ Features

- 🎲 **Random Commands**
  - `/moneta` → Testa o Croce
  - `/dado` → Dado da 6
  - `/8ball` → Risposte discutibili dal destino

- 👤 **User & Server Info**
  - `/userinfo`
  - `/serverinfo`
  - `/avatar`

- 🧹 **Moderazione**
  - `/pulisci` → elimina messaggi (permessi richiesti)

- 💬 **Fun**
  - `/schiaffeggia`
  - `/random` → immagini random
  - `/dire` → fai parlare il bot (solo owner 👀)

- 🌐 **Web Dashboard**
  - Stato bot (Online/Offline)
  - Latenza live
  - Endpoint `/ping` (JSON)
